# BP Tracker PWA

A minimalist, serverless Progressive Web App for frictionless blood pressure logging. 

Designed for daily routines, it eliminates the need for a traditional database or app store deployment. It features a native-like wheel picker UI for rapid data entry and relies on a zero-maintenance architecture: the frontend is statically hosted on GitHub Pages, while data is POSTed directly to a private Google Sheet via Google Apps Script. 

Fast, offline-capable, and keeps you in 100% control of your personal data.
